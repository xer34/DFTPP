Created by an absent minded gamer, BattleTimer is your mobile Warhammer 40k Companion.

Currently it's features include:

  • Default time is 90 minute player turns. This can be changed in the settings if you input your total game length in minutes. For example a 2:30 hour round would be input as a 180 minutes.
  • Clock will automatically pause when turn is ended.
  • Clock can be passed back and forth between players using the Overwatch button when the game begins. The current players time will automatically pause when this is pressed.
  • Checklists can be edited prior to the game under settings.
  • Pregame will disappear after the first battle round.
  • Checklists do not have a maximum number of items, however only 9 are visible at a time.
  • Plus and minus will add and subract. I know. Super exciting.
  • On soft reset of game state, these counters will reset to zero.
  • Uses easy to read fonts, colors and styles to make things easy on the eyes, also uses black where possible to save on battery life.
  • Anonymous data is collected in regards to timers, list items and amount of time spent in phases.
  • This data is used for meta analysis and for me to get a better handle on how to structure the list database.
  • In the future lists will be uploadable and downloadable. Currently I'm still working on a way of doing this and am still figuring out how to implement this feature.
